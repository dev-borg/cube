
§ git log

- git-log(1)
	° show commit logs

- examples
	° git log
	° git log -n: Show the last n commits.
	° git log --oneline: Show a simplified one-line summary of each commit.
	° git log --author=<author>: Show only commits by the specified author.
	° git log --since=<date>: Show commits since the specified date.
	° git log --grep=<pattern>: Show only commits whose messages match the specified pattern.

- git log: tree-like output
	° git log --graph --oneline --all
