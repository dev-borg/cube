
§ origin

- "origin" 
	° refers to the default remote repository associated with your local Git repository
	° when you clone a repository, Git automatically sets up a remote called "origin" to point to the URL from which you cloned the repository
	° eliminates need to specify the remote and branch ever time a "push" or "pull" is performed

- example
	° git pull -u origin <branch>
	  -u (--set_upstream)
	  set current branch's upstream to specified branch
